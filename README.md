如果您想直接访问和使用该系统，您可以访问：http://59.110.174.202:8090/

## 以下为部署本仓库代码的相关说明
## Build Setup

```bash
# clone the project
git clone https://github.com/yangyixuan327/pet_hospital_admin.git

# enter the project directory
cd pet_hospital_admin

# install dependency
npm install

# develop
npm run dev
```

This will automatically open http://localhost:9528

## Build

```bash
# build for test environment
npm run build:stage

# build for production environment
npm run build:prod
```

## Advanced

```bash
# preview the release environment effect
npm run preview

# preview the release environment effect + static resource analysis
npm run preview -- --report

# code format check
npm run lint

# code format check and auto fix
npm run lint -- --fix
```
